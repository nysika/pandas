## 🎯 Описание бизнес-требований

Ваш отдел маркетинга хочет получить **агрегированный отчёт**, который поможет:

| № | Требование | Описание |
|---|------------|----------|
| 1 | **Оценить реальную эффективность** | Каждого товара с учётом скидок |

## 💻 Техническая задача для программиста

Используя `pandas` (и при необходимости `numpy`, `datetime`, `itertools`), необходимо:

### 1. Загрузить данные
- Загрузить `sales_data.csv`

### 2. Рассчитать для каждой строки

```python
revenue = unit_price * quantity_sold * (1 - discount)
cost_price = unit_price * 0.6  # для всех товаров
profit = revenue - cost_price * quantity_sold
3. Сгруппировать данные по product_id и создать признаки
Признак	Описание
total_quantity	Общее количество продаж
total_revenue	Общая выручка
avg_discount	Средняя скидка
unique_regions	Количество разных регионов, где продавался товар
sales_frequency	Количество дней с продажами этого товара
4. Выполнить региональный анализ
Для каждого region найти топ-2 товара по сумме profit и сохранить в отдельном DataFrame.

Определить регионы, где profit от сегмента Премиум составляет более 40% от общего profit региона.

5. Построить прогноз (упрощённый, но правдоподобный)
Для каждого товара экстраполировать продажи на следующие 7 дней, используя линейную регрессию (через sklearn или numpy.polyfit) по дням исходных продаж.

Если для товара меньше 3 уникальных дней продаж — использовать медианное значение quantity_sold за все дни.

6. Применить кластеризацию (k-means, через sklearn) для товаров
Признаки для кластеризации:

total_quantity

total_revenue

avg_discount

unique_regions

Полученный прогноз на 7 дней (как дополнительный вес)

Условия:

Количество кластеров определить методом локтя (Elbow method)

Результат — новый столбец cluster_id в итоговой таблице товаров

7. Создать итоговый отчёт — Excel-файл
